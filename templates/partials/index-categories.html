WildFly Capabilities

A central location for the formal description of the capabilities provided by a WildFly process

WildFly capabilities are described in capability documents in the capabilities registry project. Each document contains information on how they are named, the service value type or runtime API they expose, other capabilities they require, and other information about their use.

Capabilities can affect one or many categories. This page lists all categories and their respective capabilities.
